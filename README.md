## cloudflare worker部署命令

[具体部署操作可以观看视频教学](https://www.youtube.com/watch?v=cthl7LLbTv0&t=6s)

创建项目

>npm create cloudflare@latest

安装依赖js-yaml

> npm install js-yaml

测试

> wrangler dev

部署到cloudflare

> wrangler deploy
